
Galleon provisioning tool.

Usage: <galleon.sh|.bat> [options] [command]

To execute a command: <galleon.sh|.bat> <command>
eg: galleon.sh install wildfly:current --dir=../latest-wildfly

To enter interactive console: <galleon.sh|.bat>
